# Finly 개발 기록

## 2차 개발

### 1단계 — 시장(Markets) 페이지 재구성

> 데이터 확장 핵심

- 탭 UI 구성 (지수 / 선물 / 원자재 / 채권 / 주식)
- 선물 API + 카드
- 원자재 API + 카드
- 채권 API + 수익률 곡선 차트
- 주요국 주식 API + 카드 (미국 / 한국 / 일본)

**상태:** 완료

---

### 2단계 — 대시보드 재구성

> Markets 완성 후 요약 발췌

- 암호화폐 BTC / ETH 2개로 축소
- 선물 / 원자재 요약 추가
- 주요국 주식 탭 추가

**상태:** 완료

---

### 3단계 — 포트폴리오 고도화

> DB 설계 필요, 독립적

- 스냅샷 테이블 설계 (Supabase)
- 자산 추이 차트
- UI/UX 개선

**상태:** 진행 중
